Dovetail is a convention-driven application wiring library for .NET. It uses a Roslyn source generator to resolve and order conventions at build time — eliminating runtime reflection and making your app AOT and trimming safe.

> [!NOTE]
> Dovetail requires .NET 8 or .NET 10 and a compatible Roslyn-based build.
